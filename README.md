# 🚀 Intellexa – AI Based Decision Support System for Business

Intellexa is a powerful **AI-driven business analytics platform** that helps businesses transform raw data into meaningful insights, predictions, and smart decisions.

It allows users to upload **Excel/CSV data**, analyze performance, detect risks, and get **AI-powered recommendations** — all in a simple dashboard.

---

## 🌐 Live Demo

🔗 https://intellexaa.netlify.app  

---

## 📌 Features

### 📊 Data Upload & Processing
- Upload business data using **CSV or Excel files**
- Automatic data parsing and preprocessing

### 📈 Data Visualization
- Interactive charts:
  - Line Charts
  - Bar Charts
  - Pie Charts
  - Area Charts
- KPI metrics:
  - Total Revenue
  - Customer Count
  - Average Order Value
  - Retention Rate
  - Revenue Growth

### 🤖 AI-Powered Insights
- Automatically generated:
  - Business insights
  - Risk detection
  - Recommendations

### 🔮 Predictive Analytics
- Forecast future revenue using **Machine Learning**
- 30-day prediction graph

### 💬 AI Assistant
- Ask business-related questions
- Get intelligent answers using LLM

### 🔐 Authentication & Security
- JWT-based authentication
- Secure cookies
- Protected routes
